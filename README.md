# Spotify Notetaker

A beautiful React demo app for taking notes on your favorite Spotify tracks.

![Spotify Notetaker](https://img.shields.io/badge/React-18.3-blue) ![Vite](https://img.shields.io/badge/Vite-6.0-purple)

## Features

- 🎵 Browse your music library
- 📝 Take notes on individual tracks
- 🔍 Search tracks and artists
- 💾 Notes persist during session
- 🎨 Modern, Spotify-inspired UI

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/sergiopesch/Spotify-Notetaker.git
cd Spotify-Notetaker
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:5173](http://localhost:5173) in your browser.

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint

## Tech Stack

- **React 18** - UI library
- **Vite** - Build tool and dev server
- **CSS3** - Styling with modern features (gradients, backdrop-filter)

## Demo

This is a demo application with sample data. In a real implementation, you would:

1. Integrate with the Spotify Web API for real track data
2. Add user authentication via Spotify OAuth
3. Persist notes to a backend database
4. Add features like timestamps during playback, sharing notes, etc.
